# Assets Directory — LiveStream Premium

This directory contains all static assets for the application.

## Required Assets (User Must Provide)

### Fonts (assets/fonts/)
- `Inter-Regular.ttf` — Regular weight body text
- `Inter-Medium.ttf` — Medium weight for emphasis
- `Inter-SemiBold.ttf` — Semi-bold for headings
- `Inter-Bold.ttf` — Bold for titles
- `JetBrainsMono-Regular.ttf` — Monospace for code/timer displays

Download Inter font: https://fonts.google.com/specimen/Inter
Download JetBrains Mono: https://fonts.google.com/specimen/JetBrains+Mono

### DeepAR Face Masks (assets/masks/)
Place your `.deepar` format face mask files in the appropriate directories:
- `masks/male/` — Male face swap masks (.deepar files)
- `masks/female/` — Female face swap masks (.deepar files)

Get masks from: https://developer.deepar.ai/asset-store
Or create custom masks with: DeepAR Studio

### Images (assets/images/)
- `app-icon.png` — 1024x1024 app icon (PNG)
- `splash.png` — Splash screen image
- `adaptive-icon.png` — Android adaptive icon foreground
- `favicon.png` — Web favicon
- `default-stream-thumb.png` — Default stream thumbnail placeholder
- `default-mask.png` — Default mask thumbnail placeholder
